## Run Example

python3 -m pip install -r requirements.txt

## Run Format
Parameters: source_conf, target_conf, output_img, lambda_float
```bash
python3 ./homography_2020.py -s path/to/source/conf -t path/to/target/conf -o path/to/output/image -l 0.0
```
